# 📦 Inventory Management System

## 📖 Description

The **Inventory Management System** is a **Core Java** console-based application that enables efficient management of product inventory using **Object-Oriented Programming (OOP)** and the **Java Collections Framework**. It provides a menu-driven interface to perform CRUD operations, track stock levels, and generate inventory reports in **PDF format** using the **iTextPDF** library.

---

## 🚀 How It Works

The application allows users to manage inventory through a simple console interface. Users can add, update, search, delete, and view products while maintaining accurate stock information using Java Collections. The system also generates a well-formatted **PDF report** containing the complete inventory details for easy viewing, sharing, and record maintenance.

### ⚙️ Workflow

### ▶️ 1. Launch Application
- Start the application.
- Select an operation from the main menu.

### ➕ 2. Add Product
- Enter **Product ID, Name, Category, Price, Quantity,** and **Supplier**.
- Store the product in the inventory.

### 📋 3. View Products
- Display all available products.
- Show complete inventory details.

### 🔍 4. Search Product
- Search by **Product ID** or **Product Name**.
- Display matching product information.

### ✏️ 5. Update Product
- Modify product details such as **Price**, **Quantity**, or **Supplier**.
- Save the updated information.

### ❌ 6. Delete Product
- Remove products from the inventory.
- Keep records organized and up to date.

### 📊 7. Generate PDF Report
- Export the complete inventory data to a **PDF document**.
- Reports are generated using the **iTextPDF** library for easy documentation and sharing.

### 🚪 8. Exit Application
- Safely terminate the application.
