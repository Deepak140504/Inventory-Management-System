📦 Inventory Management System

🚀 How It Works
The Inventory Management System is a Core Java console-based application that helps users efficiently manage inventory records. It provides an interactive menu-driven interface to perform various inventory operations using Object-Oriented Programming (OOP) and the Java Collections Framework without requiring a database.

⚙️ Workflow
▶️ 1. Launch Application
Start the application to access the main menu.
Choose an operation by entering the corresponding option.
➕ 2. Add Product
Enter product details such as Product ID, Name, Category, Price, Quantity, and Supplier.
The product is added to the inventory.
📋 3. View Products
Display all available products with complete information.
View inventory records in an organized format.
🔍 4. Search Product
Search products using Product ID or Product Name.
Instantly retrieve matching product details.
✏️ 5. Update Product
Modify existing product details like Price, Quantity, or Supplier.
Updated information is saved immediately.
❌ 6. Delete Product
Remove unwanted products from the inventory.
Keep inventory records clean and up to date.
📊 7. Stock Management
Monitor available stock quantities.
Maintain accurate inventory levels after every operation.
🚪 8. Exit Application
Safely close the application after completing all tasks.

🎯 Project Objective
To develop a simple and efficient Inventory Management System that demonstrates Core Java, OOP concepts, and the Java Collections Framework while enabling users to manage products, monitor stock, and perform CRUD operations through a user-friendly console interface.
